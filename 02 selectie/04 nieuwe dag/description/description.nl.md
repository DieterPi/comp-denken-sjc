## Opgave

Schrijf een programma dat de huidige dag, het huidige maandnummer en het huidige jaar vraagt. Vervolgens bepaalt je programma de volgende dag (het volgende maandnummer en het volgende jaar). Je mag ervan uitgaan dat het ingevoerde jaar **geen schrikkeljaar** is.

![Google calendar.](media/google_calendar.gif "Google calendar."){:data-caption="Google calendar." width="400px"}

#### Voorbeelden

Indien `1` januari (maand `1`) `2025` ingegeven wordt, verschijnt er:

```
Morgen is het 2 / 1 / 2025
```

Bij de invoer `31` december (maand `12`) `2025`, verschijnt er:

```
Morgen is het 1 / 1 / 2026
```
