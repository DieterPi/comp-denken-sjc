## Opgave
Lees een reeks getallen in waarbij de invoer stopt indien de gebruiker `0` intikt.

Geef daarna op het scherm weer hoeveel priemgetallen ingetikt werden.

#### Voorbeeld

Indien de gebruiker achtereenvolgend `5`, `2`, `4`, `6`, `1`, `7`, `10` en tot slot `0` ingeeft, dan verschijnt er:
```
Er werden 3 priemgetallen ingevoerd.
```
Want de getallen `5`, `2` en `7` zijn immers priem.
