De rij van Göbel wordt via dit voorschrift gedefinieerd:

$$\mathsf{u_n = \dfrac{u_0^2 + u_1^2 + \ldots + u_{n-1}^2}{n-1}, \qquad\text{met}, \qquad u_0 = 1 = u_1}$$

Men krijgt dus als rij:

$$
\mathsf{1, \qquad 1, \qquad 2, \qquad 3, \qquad 5, \qquad 10,\qquad \ldots }
$$

## Opgave
Schrijf een programma dat een rangnummer `n` aan de gebruiker vraagt. Daarna berekent je programma het n<sup>de</sup> getal in de rij van Göbel en toont het dit op het scherm.

#### Voorbeelden

Bij invoer `4` verschijnt:

```
Het 4e getal is 3
```

Bij invoer `7` verschijnt:

```
Het 7e getal is 28
```

Bij invoer `1` verschijnt:

```
Het 1e getal is 1
```
