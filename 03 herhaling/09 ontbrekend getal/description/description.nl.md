
## Opgave

Schrijf een programma dat verschillende natuurlijke getallen vraagt aan de gebruiker. Alle getallen gesorteerd van klein naar groot zouden een opeenvolgende rij vormen (te beginnen vanaf `1`), maar **één getal ontbreekt**. Het programma stopt indien de gebruiker `-1` intikt.

Bepaal welk getal niet werd ingetikt...

#### Voorbeelden

Indien de gebruiker achtereenvolgend `2`, `1`, `4` en daarna `-1` intikt, dan verschijnt er:
```
Het getal 3 ontbreekt.
```

Indien de gebruiker achtereenvolgend `8`, `3`, `7`, `1`, `2`, `5`, `4` en daarna `-1` intikt, dan verschijnt er:
```
Het getal 6 ontbreekt.
```

Indien de gebruiker achtereenvolgend `1`, `2`, `3`, `4` en daarna `-1` intikt, dan verschijnt er:
```
Het getal 5 ontbreekt.
```
