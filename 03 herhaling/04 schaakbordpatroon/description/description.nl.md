## Opgave

Vraag aan de gebruiker het aantal rijen en het aantal kolommen en druk nadien een schaakbordpatroon af waarbij het teken linksboven een `X` is. In plaats van een spatie druk je een underscore `_` af.

#### Voorbeeld

Indien de gebruiker `6` en `8` intikt, verschijnt er:
```
X_X_X_X_
_X_X_X_X
X_X_X_X_
_X_X_X_X
X_X_X_X_
_X_X_X_X
```
